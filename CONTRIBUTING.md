# Contributing data

新增条目时建议提供：

1. 系统名称与别名。
2. Lensed QSO / Lensed SN 分类。
3. lens/source redshift。
4. 像数与 galaxy-scale / cluster-scale 分类。
5. 发现论文、建模论文、时延论文、数据论文。
6. 数据下载入口，例如 MAST、OSC、TNS、CASTLES、DES、survey cutout。
7. 缩略图来源和 image credit。
8. 是否有 time delay、kinematics、LOS/environment、microlensing 相关结果。

请优先引用原论文或官方数据库，不建议只引用二级网页。
