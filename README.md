# Show Me the Lens

Data catalogue of gravitationally lensed quasars and supernovae.

## Current scope

- 24 lensed QSO entries and 13 lensed SN entries.
- Bilingual UI with strict Chinese-only and English-only display modes.
- MAST-first data links using source coordinates.
- Reference links tagged as first report, modelling, follow-up, review/sample, or candidate/discovery.
- Pagination at 12, 24, or 48 sources per page.

Real thumbnails can be swapped into `assets/img/thumbs/` when the generated panels from `raw_lens_panels_wcs` are available locally.
